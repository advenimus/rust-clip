RustClip

Self-hosted, end-to-end encrypted universal clipboard sync — pure Rust from the server to the desktop.

Text, images, and files sync live across macOS, Windows, and Linux devices tied to the same account. The server routes encrypted blobs and never sees plaintext: content keys are derived from the user's password via Argon2id and every payload is sealed with XChaCha20-Poly1305 before it hits the wire.
